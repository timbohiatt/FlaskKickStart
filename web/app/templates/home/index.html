Welcome to the Tim's Base Web App

The whole purpose of thi Docker Compose Image was to get you going with a Web App Quickly! Hopefully, all going well you should have the following:

  • [FLASK] Base Flask Web Application
    • Implements Flask
    • Implements Blueprint Factory (Default with Home & Admin Blueprints
    • SQL Alchemy Backend Connected to MySQL Container from Docker Compose
    • Celery Background Tasks With an Example in the Home/Views Module
    • Bootstrap CSS and Framework for Quick Formatting of your Site
  • [MYSQL] MySQL Container
    • MySQL Running in Container with link to Flask App through SQL Alchemy
  • [CELERY] Celery Container
    • Celery Container Linked to the Flask Web App, Ready to use through the Rabbit MQ Broker
  • [RABBIT-MQ] Rabbit-MQ Container
    • Rabbit-MQ Message Broker configured to listen to messages between Flask Web App and the Celery Broker

Really hope you find this a helpful way to get going with your web application within a container application.
